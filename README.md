# Weather API Integration

## Description
This project is a Python-based Weather Application that fetches real-time weather information for a user-specified city using the wttr.in API.

## Features
- Fetches real-time weather data
- Displays temperature, humidity, and weather conditions
- Uses Python requests library
- Handles API and network errors

## Technologies Used
- Python
- Requests Library
- wttr.in API

## How to Run
1. Install Python.
2. Install the requests library:
   pip install requests
3. Run the program:
   python weather.py
4. Enter a city name when prompted.

## Example

Input:
Hyderabad

Output:
Weather Report
City: Hyderabad
Temperature: 32 °C
Humidity: 58 %
Condition: Partly cloudy
